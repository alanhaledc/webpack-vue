# webpack-vue

> 使用 Webpack 搭建 Vue3 开发环境

[**vue2 开发环境**](https://github.com/haledc/webpack-vue/tree/vue2)

## 实现功能

- ES6
- Vue3
- Scss/css (Modules + AutoPrefixer)
- Lint (ESLint + Prettier)

## 启动

```bash
# install node_modules
yarn
# or npm install

# development env
yarn dev

# production env
yarn build
yarn start
```
